# 📁 Raw Data Guide - SNF Dashboard

## 📌 데이터 구조 개요

이 프로젝트는 **노션에서 내보낸 CSV 파일**을 원본 데이터로 사용합니다.
프로젝트 루트 폴더에 아래 파일들이 있어야 합니다.

```
2025년 6월 SNF 조사/
├── TOP10 게임 종합 평가 [NOTION_ID].csv      ← TOP10 상세 정보
├── TOP10 차트인 횟수 [NOTION_ID].csv         ← 차트인 횟수 요약
├── 가장 많이 플레이한 TOP50 게임 [NOTION_ID].csv  ← TOP50 전체 목록
├── 결산 페이지 [NOTION_ID].csv               ← 커뮤니티/업데이트 상세
├── 떠오르는 출시 예정 게임 [NOTION_ID].csv    ← 떠오르는 게임 차트
├── 인기 체험판 [NOTION_ID].csv               ← 인기 체험판 차트
├── 인기 출시 예정 게임 [NOTION_ID].csv        ← 인기 출시 예정 차트
├── 전체 장르 - 각 게임별 SNF기간 3종 차트인 횟수 [NOTION_ID].csv  ← 통합 차트
├── scripts/
│   └── generate_insights.py                 ← AI 인사이트 생성 스크립트
└── github_data/
    ├── 01_executive/                        ← 생성된 인사이트 저장
    ├── 02_top_games/
    ├── 03_charts/
    └── 04_report/
```

## 🔄 데이터 업데이트 흐름

```
1. 노션에서 CSV 내보내기
   └─ 각 데이터베이스를 CSV로 Export
   
2. 프로젝트 루트에 파일 배치
   └─ 기존 파일 덮어쓰기 OK (파일명 패턴만 유지)
   
3. AI 스크립트 실행
   └─ python scripts/generate_insights.py
   
4. 인사이트 생성 완료
   └─ github_data/ 폴더에 새 CSV 저장
   
5. GitHub에 Push
   └─ git add . && git commit -m "Update" && git push
   
6. 대시보드 자동 업데이트
   └─ 브라우저 새로고침하면 반영
```

## ⚠️ 주의사항

1. **파일명 패턴 유지**: 노션에서 내보낼 때 파일명 앞부분은 동일해야 함
   - ✅ `TOP10 게임 종합 평가 215dadb56b2f80ccac70c9987aa5ea5c.csv`
   - ✅ `TOP10 게임 종합 평가 abcd1234.csv` (ID가 달라도 OK)
   - ❌ `게임 종합 평가.csv` (패턴 불일치)

2. **인코딩**: UTF-8로 저장되어야 함 (노션 기본값 OK)

3. **구분자**: 쉼표(,)로 구분된 CSV (노션 기본값 OK)

4. **날짜 형식**: 한국어 날짜 형식 사용 (`2025년 6월 10일`)
