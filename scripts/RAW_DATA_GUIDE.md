# 📁 Raw Data Guide - SNF Dashboard (v2.0)

## 📌 데이터 구조 개요

이 프로젝트는 **노션에서 내보낸 CSV 파일**을 원본 데이터로 사용합니다.
`github_data/raw/` 폴더에 아래 파일들이 있어야 합니다.

```
2025년 6월 SNF 조사/
├── github_data/
│   ├── raw/                                    ← 📂 원본 데이터 (노션 CSV)
│   │   ├── TOP10 게임 종합 평가 [NOTION_ID].csv
│   │   ├── TOP10 차트인 횟수 [NOTION_ID].csv
│   │   ├── 가장 많이 플레이한 TOP50 게임 [NOTION_ID].csv
│   │   ├── 결산 페이지 [NOTION_ID].csv
│   │   ├── 떠오르는 출시 예정 게임 [NOTION_ID].csv
│   │   ├── 인기 체험판 [NOTION_ID].csv
│   │   ├── 인기 출시 예정 게임 [NOTION_ID].csv
│   │   └── 전체 장르 - 각 게임별 SNF기간 3종 차트인 횟수 [NOTION_ID].csv
│   │
│   ├── 01_executive/                           ← 📊 AI 생성 (7개 CSV)
│   ├── 02_top_games/                           ← 📊 AI 생성 (6개 CSV)
│   ├── 03_charts/                              ← 📊 AI 생성 (7개 CSV)
│   ├── 04_report/                              ← 📊 AI 생성 (5개 CSV)
│   └── prompts/                                ← 📝 AI 프롬프트
│
├── scripts/
│   ├── generate_insights.py                    ← 🤖 AI 인사이트 생성 스크립트
│   ├── .env                                    ← 🔑 API 키 (GitHub 제외)
│   ├── HOW_TO_RUN.md
│   └── RAW_DATA_GUIDE.md (이 파일)
│
└── dashboard.html                              ← 🌐 대시보드
```

## 🔄 데이터 업데이트 흐름

```
1. 노션에서 CSV 내보내기
   └─ 각 데이터베이스를 CSV로 Export
   
2. github_data/raw/ 폴더에 파일 배치
   └─ 기존 파일 덮어쓰기 OK (파일명 패턴만 유지)
   
3. AI 스크립트 실행
   └─ cd scripts && python generate_insights.py
   
4. 인사이트 생성 완료 (25개 CSV)
   └─ github_data/01~04 폴더에 새 CSV 저장
   
5. GitHub에 Push
   └─ git add github_data/ && git commit -m "Update" && git push
   
6. 대시보드 자동 업데이트
   └─ 브라우저 새로고침하면 반영
```

## ⚠️ 주의사항

### 파일 위치
- ✅ `github_data/raw/TOP10 게임 종합 평가 xxxxxx.csv`
- ❌ 프로젝트 루트에 직접 배치 (X)

### 파일명 패턴
- ✅ `TOP10 게임 종합 평가 215dadb56b2f80ccac70c9987aa5ea5c.csv`
- ✅ `TOP10 게임 종합 평가 abcd1234.csv` (ID가 달라도 OK)
- ❌ `게임 종합 평가.csv` (패턴 불일치)

### 인코딩
- UTF-8로 저장 (노션 기본값 OK)
- AI가 생성하는 CSV는 UTF-8 BOM (Excel 호환)

### 구분자
- 쉼표(,)로 구분된 CSV (노션 기본값 OK)

### 날짜 형식
- 한국어 날짜 형식 사용 (`2025년 6월 10일`)

## 📞 문제 발생 시

1. **파일을 찾을 수 없습니다**
   - `github_data/raw/` 폴더 확인
   - 파일명 패턴 확인

2. **데이터가 비어있습니다**
   - 노션에서 다시 내보내기
   - CSV 파일 내용 확인

3. **한글 깨짐**
   - 노션 CSV는 UTF-8이므로 정상
   - Excel에서 열 때: 데이터 → 텍스트 가져오기 → UTF-8 선택
