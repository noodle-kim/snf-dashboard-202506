# 📈 차트 성과 분석 - AI 인사이트 프롬프트

## 📋 개요
이 프롬프트는 대시보드의 **세 번째 페이지 "차트 성과 분석"**에 표시될 AI 인사이트를 생성합니다.

---

## 🎯 인사이트 생성 목표
3종 차트(인기 체험판, 인기 출시 예정, 떠오르는 출시 예정) 데이터를 분석하여 **차트 진입 및 유지 전략 4개**를 도출합니다.

---

## 📊 입력 데이터 (raw/03_charts/)

| 파일명 | 내용 | 활용 방법 |
|--------|------|----------|
| `01_kpi_cards.csv` | 차트 진입 게임 수, 최다 차트인 게임 정보 | 전체 차트 성과 요약 |
| `03_chart_data.csv` | 차트 종류별 분포, 차트인 횟수 TOP 10, 일자별 활동 | 차트 패턴 분석 |
| `05_demo_chart.csv` | 인기 체험판 차트 상세 (게임명, 등장 횟수, 최고 순위) | 체험판 차트 전략 |
| `06_popular_upcoming.csv` | 인기 출시 예정 차트 상세 | 출시 예정 차트 전략 |
| `07_trending_upcoming.csv` | 떠오르는 출시 예정 차트 상세 | 신흥 게임 차트 전략 |

---

## 📝 프롬프트

```
당신은 Steam Next Fest 차트 분석 전문가입니다.

아래 3종 차트 데이터를 분석하여 **개발사가 SNF 차트에 진입하고 유지하기 위한 전략적 인사이트 4개**를 도출해주세요.

### 분석할 데이터:
[여기에 01_kpi_cards.csv, 03_chart_data.csv, 05_demo_chart.csv, 06_popular_upcoming.csv, 07_trending_upcoming.csv 내용 삽입]

### 3종 차트 설명:
1. **인기 체험판**: 체험판 다운로드/플레이 기반 순위
2. **인기 출시 예정**: 찜 목록 추가 기반 순위
3. **떠오르는 출시 예정**: 급상승 관심도 기반 순위

### 출력 형식:
정확히 아래 CSV 형식으로 출력해주세요. 다른 텍스트 없이 CSV만 출력하세요.

id,icon,title,description,color
1,,{제목1},{설명1},#0047AB
2,,{제목2},{설명2},#0047AB
3,,{제목3},{설명3},#0047AB
4,,{제목4},{설명4},#3B82F6

### 분석 포인트:
1. **가장 많이 차트에 오른 게임**의 특징 (예: 빈딕투스 14회)
2. **5일 연속 차트 유지** 게임들의 공통점
3. **차트 종류별 특성** (인기 체험판 vs 출시 예정 vs 떠오르는)
4. **차트 진입 최소 기준** (몇 회 이상이면 성공?)
5. **급상승/급하락** 패턴 분석

### 인사이트 작성 가이드라인:
1. **title**: 7단어 이내, 전략적 제목 (예: "10회 이상 진입 = 성공")
2. **description**: 구체적 수치와 함께 실행 가능한 전략 제시
3. 개발사 규모(인디/AA/AAA)에 맞는 차별화된 조언 포함
4. **color**: 처음 3개는 #0047AB, 마지막은 #3B82F6
```

---

## 📤 출력 파일
- **파일명**: `insights/03_charts_insights.csv`
- **형식**: CSV (id, icon, title, description, color)

## ✅ 출력 예시

```csv
id,icon,title,description,color
1,,상위 3개 게임이 주도,빈딕투스 나혼자레벨업 Jump Ship 3개 게임이 전체 차트 노출의 25%를 차지했습니다.,#0047AB
2,,체험판이 가장 효과적,인기 체험판 차트가 전체 노출의 40%를 차지해 가장 효과적인 홍보 채널로 확인됐습니다.,#0047AB
3,,10회 이상 진입 = 성공,10회 이상 차트에 오른 게임은 4개뿐입니다. 이 기준을 달성하면 성공적인 SNF라고 볼 수 있습니다.,#0047AB
4,,3종 차트 동시 공략이 핵심,세 종류의 차트에 모두 진입한 게임들이 평균 10회 이상 노출되며 높은 성과를 기록했습니다.,#3B82F6
```
