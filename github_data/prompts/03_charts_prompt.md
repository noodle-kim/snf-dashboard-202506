# 📈 차트 성과 분석 - AI 인사이트 프롬프트

## 📋 개요
이 프롬프트는 대시보드의 **세 번째 페이지 "차트 성과 분석"**에 표시될 AI 인사이트를 생성합니다.

---

## 🎯 생성 목표
3종 차트(인기 체험판, 인기 출시 예정, 떠오르는 출시 예정) 데이터를 분석하여 **7개의 CSV 파일**을 생성합니다:
1. `01_kpi_cards.csv` - 차트 성과 KPI 3개
2. `02_key_findings.csv` - 차트 분석 인사이트 4개
3. `03_chart_data.csv` - 차트별 통계 데이터
4. `04_strategy_cards.csv` - 차트 진입 전략 3개
5. `05_demo_chart.csv` - 인기 체험판 차트 상세
6. `06_popular_upcoming.csv` - 인기 출시 예정 차트 상세
7. `07_trending_upcoming.csv` - 떠오르는 출시 예정 차트 상세

---

## 📊 입력 데이터 (raw/ 폴더)

| 파일명 | 내용 | 활용 |
|--------|------|------|
| `전체 장르 - 각 게임별 SNF기간 3종 차트인 횟수*.csv` | 게임명, 날짜, 랭킹, 차트 구분 | 모든 차트 분석 |
| `인기 체험판*.csv` | 인기 체험판 차트 데이터 (있다면) | 체험판 차트 분석 |
| `인기 출시 예정 게임*.csv` | 인기 출시 예정 차트 데이터 (있다면) | 출시 예정 분석 |
| `떠오르는 출시 예정 게임*.csv` | 떠오르는 출시 예정 데이터 (있다면) | 신흥 게임 분석 |

## 📝 프롬프트 1: KPI 카드 (01_kpi_cards.csv)

```
3종 차트인 횟수 데이터를 분석하여 **핵심 KPI 3개**를 추출해주세요.

### 출력 형식 (CSV):
id,icon,value,label,description,color
1,📊,{값},{지표명},{설명},#0047AB
2,🎯,{값},{지표명},{설명},#10B981
3,🎮,{값},{지표명},{설명},#8B5CF6

### 필수 KPI:
1. 총 차트 진입 게임 수 (중복 제외 고유 게임)
2. 최다 차트인 게임 (예: "Jump Ship 15회")
3. 체험판 차트 비율 (인기 체험판 ÷ 전체 노출)

### 계산 방법:
- 고유 게임 수: 게임명 기준 중복 제거
- 차트인 횟수: 게임별 출현 횟수 합계
- 체험판 비율: 차트 구분="인기 체험판" 건수 ÷ 전체 건수
```

## 📝 프롬프트 2: 핵심 발견점 (02_key_findings.csv)

```
차트 데이터를 분석하여 **개발사가 알아야 할 핵심 발견점 4개**를 도출해주세요.

### 출력 형식 (CSV):
id,icon,title,description,color
1,📈,{제목},{상세 설명 50자 이상},#0047AB
2,🎮,{제목},{상세 설명 50자 이상},#0047AB
3,🎯,{제목},{상세 설명 50자 이상},#0047AB
4,🚀,{제목},{상세 설명 50자 이상},#3B82F6

### 분석 포인트:
1. **가장 많이 차트에 오른 게임**의 특징 (예: 빈딕투스 14회)
2. **연속 차트 유지** 게임들의 공통점
3. **차트 종류별 특성** (인기 체험판 vs 출시 예정 vs 떠오르는)
4. **차트 진입 최소 기준** (몇 회 이상이면 성공?)

### 인사이트 작성 규칙:
- 반드시 구체적 수치 포함 (예: "상위 3개 게임이 전체의 25%")
- 실행 가능한 전략 제시
```

## 📝 프롬프트 3: 차트별 통계 (03_chart_data.csv)

```
3종 차트의 통계 데이터를 생성해주세요.

### 출력 형식 (CSV):
chart_type,stat_type,label,value,percentage
인기 체험판,count,총 노출 횟수,{값},{비율%}
인기 출시 예정 게임,count,총 노출 횟수,{값},{비율%}
떠오르는 출시 예정 게임,count,총 노출 횟수,{값},{비율%}
top_games,{게임명},차트인 횟수,{값},{비율%}

### 분석 항목:
1. 차트 종류별 노출 횟수 분포
2. 차트인 횟수 TOP 10 게임
3. 일자별 차트 진입 패턴 (6/10~6/16)
```

## 📝 프롬프트 4: 차트 진입 전략 (04_strategy_cards.csv)

```
차트 데이터 분석을 바탕으로 **차트 진입 전략 3개**를 도출해주세요.

### 출력 형식 (CSV):
id,icon,title,description,details
1,🎯,{전략 제목},{한줄 설명},{세부내용1}|{세부내용2}|{세부내용3}
2,📊,{전략 제목},{한줄 설명},{세부내용1}|{세부내용2}|{세부내용3}
3,🚀,{전략 제목},{한줄 설명},{세부내용1}|{세부내용2}|{세부내용3}

### 전략 주제:
1. **체험판 차트 공략**: 인기 체험판 차트가 가장 효과적인 이유
2. **연속 노출 전략**: 3일 연속 차트 유지하는 방법
3. **3종 차트 동시 공략**: 세 차트에 모두 진입하는 전략
```
