# 🏆 TOP 게임 분석 - AI 인사이트 프롬프트

## 📋 개요
이 프롬프트는 대시보드의 **두 번째 페이지 "TOP 게임 분석"**에 표시될 AI 인사이트를 생성합니다.

---

## 🎯 인사이트 생성 목표
TOP 10 / TOP 50 게임 데이터를 분석하여 **상위권 게임들의 공통 성공 요인 4개**를 도출합니다.

---

## 📊 입력 데이터 (raw/02_top_games/)

| 파일명 | 내용 | 활용 방법 |
|--------|------|----------|
| `01_kpi_cards.csv` | TOP 10 총 찜 수 증가, 1위 게임 성과 | 전체 성과 규모 파악 |
| `03_top10_table.csv` | TOP 10 게임별 장르, 리뷰, 찜 증가, 차트인 횟수 | 상위 게임 상세 분석 |
| `04_top10_charts.csv` | 찜 수 차트, 리뷰 분포, 장르 분포 데이터 | 시각화용 데이터 |
| `05_top50_table.csv` | TOP 50 게임의 장르, 체험판, 플레이 타입, 출시일 | 넓은 범위 트렌드 파악 |
| `06_top50_charts.csv` | TOP 50 장르 분포, 멀티/싱글 비율, 체험판 비율 | 전체 트렌드 시각화 |

---

## 📝 프롬프트

```
당신은 Steam Next Fest 게임 분석 전문가입니다.

아래 TOP 10 및 TOP 50 게임 데이터를 분석하여 **상위권 게임들의 공통 성공 요인 4개**를 도출해주세요.

### 분석할 데이터:
[여기에 01_kpi_cards.csv, 03_top10_table.csv, 05_top50_table.csv, 06_top50_charts.csv 내용 삽입]

### 출력 형식:
정확히 아래 CSV 형식으로 출력해주세요. 다른 텍스트 없이 CSV만 출력하세요.

id,icon,title,description,border_color
1,,{제목1},{설명1},#0047AB
2,,{제목2},{설명2},#3B82F6
3,,{제목3},{설명3},#F59E0B
4,,{제목4},{설명4},#8B5CF6

### 분석 포인트:
1. **체험판 제공 여부**와 성과의 상관관계
2. **멀티플레이 vs 싱글플레이** 비율 및 성과 차이
3. **리뷰 상태**(긍정적/복합적/확인불가)와 순위의 관계
4. **장르별 분포**에서 특이점
5. **한국 게임**(빈딕투스, 나혼자레벨업)의 성과 분석
6. **차트인 횟수**가 높은 게임들의 공통점

### 인사이트 작성 가이드라인:
1. **title**: 7단어 이내의 명확한 제목
2. **description**: 데이터에서 직접 확인 가능한 수치 포함 (예: "TOP 10 중 7개가...")
3. 개발사/퍼블리셔가 실행할 수 있는 조언 포함
4. **border_color**: 순서대로 #0047AB, #3B82F6, #F59E0B, #8B5CF6 사용
```

---

## 📤 출력 파일
- **파일명**: `insights/02_top_games_insights.csv`
- **형식**: CSV (id, icon, title, description, border_color)

## ✅ 출력 예시

```csv
id,icon,title,description,border_color
1,,체험판이 성공의 열쇠,TOP 10 중 5개 게임이 체험판 페이지 접속 가능. 체험판을 제공하면 유저 관심도가 크게 높아집니다.,#0047AB
2,,멀티플레이가 대세,TOP 10 중 7개가 멀티플레이 게임. 협동/경쟁 요소가 SNF에서 강력한 경쟁력이 됩니다.,#3B82F6
3,,긍정 리뷰가 증명,TOP 10 중 4개가 긍정적 이상 리뷰. 품질이 검증된 게임들이 상위권을 차지했습니다.,#F59E0B
4,,중국어권이 핵심,대부분의 TOP 게임 리뷰 언어 1위가 간체 중국어. 중국 시장 공략이 성공의 필수 요소입니다.,#8B5CF6
```
