# 🎯 핵심 요약 (Executive Summary) - AI 인사이트 프롬프트

## 📋 개요
이 프롬프트는 대시보드의 **첫 번째 페이지 "핵심 요약"**에 표시될 다양한 AI 인사이트를 생성합니다.

---

## 🎯 생성 목표
Steam Next Fest(SNF) 데이터를 분석하여 **7개의 CSV 파일**을 생성합니다:
1. `01_strategies.csv` - 다음 SNF 성공 전략 3개
2. `02_kpi_cards.csv` - 핵심 성과 지표 4개
3. `03_insights.csv` - 주요 발견점 인사이트 4개  
4. `04_top5_games.csv` - TOP 5 게임 정보
5. `05_chart_summary.csv` - 차트 성과 요약 3개
6. `06_genre_distribution.csv` - 장르 분포 데이터
7. `07_snf_guide.csv` - SNF 준비 가이드 체크리스트

---

## 📊 입력 데이터 (raw/ 폴더)

| 파일명 | 내용 | 활용 |
|--------|------|------|
| `TOP10 게임 종합 평가*.csv` | 순위, 리뷰, 찜 수, 국적 등 | TOP 5 게임, KPI, 인사이트 |
| `가장 많이 플레이한 TOP50*.csv` | 장르, 체험판, 멀티플레이 | 장르 분포, 전략 도출 |
| `전체 장르 - 각 게임별 SNF기간 3종 차트인 횟수*.csv` | 차트 진입 현황 | 차트 성과 요약 |
| `결산 페이지*.csv` | 태그, 언어, 커뮤니티 | SNF 준비 가이드 |

## 📝 프롬프트 1: 전략 카드 (01_strategies.csv)

```
당신은 Steam Next Fest 전략 컨설턴트입니다.

아래 데이터를 분석하여 **다음 SNF에서 성공하기 위한 핵심 전략 3개**를 도출해주세요.

### 분석할 데이터:
{TOP10 게임 종합 평가, TOP50 게임, 3종 차트인 횟수 데이터}

### 출력 형식 (CSV):
id,icon,title,description,details
1,🎮,{전략 제목},{한줄 설명},{세부내용1}|{세부내용2}|{세부내용3}|{세부내용4}
2,👥,{전략 제목},{한줄 설명},{세부내용1}|{세부내용2}|{세부내용3}|{세부내용4}
3,🌏,{전략 제목},{한줄 설명},{세부내용1}|{세부내용2}|{세부내용3}|{세부내용4}

### 전략 작성 가이드:
1. **title**: 명확한 전략 제목 (예: "체험판 필수 제공")
2. **description**: 전략의 핵심 요약 (20자 내외)
3. **details**: 파이프(|)로 구분된 4개의 구체적 실행 항목
   - 반드시 데이터 기반 수치 포함 (예: "TOP 10의 50%가 체험판 제공")
   - 실행 가능한 액션 아이템

### 전략 주제 예시:
- 체험판 전략: TOP 10의 50%가 체험판 제공, SNF 전 체험판 준비 필수
- 멀티플레이어: TOP 10의 70%가 멀티, 협동/경쟁 요소 추가
- 중국 시장: 리뷰 언어 1위가 간체 중국어, 중국어 현지화 필수
```

```
id,icon,value,label,sublabel,color
1,📊,{값},{지표명},{부가설명},#0047AB

4,📈,{값},{지표명},{부가설명},#F59E0B
```

## 📝 프롬프트 3: 주요 발견점 (03_insights.csv)

```
데이터를 분석하여 **개발사가 알아야 할 주요 발견점 4개**를 도출해주세요.

### 출력 형식 (CSV):
id,icon,title,description,border_color
1,🎮,{제목},{상세 설명 - 수치 포함 50자 이상},#0047AB
2,👥,{제목},{상세 설명 - 수치 포함 50자 이상},#3B82F6
3,⭐,{제목},{상세 설명 - 수치 포함 50자 이상},#F59E0B
4,🌏,{제목},{상세 설명 - 수치 포함 50자 이상},#8B5CF6

### 인사이트 작성 규칙:
1. **title**: 7단어 이내 임팩트 있는 제목 (예: "체험판이 성공의 열쇠")
2. **description**: 
   - 반드시 구체적 수치 포함 (예: "TOP 10 중 5개 게임이...")
   - 왜 중요한지 이유 설명
   - 개발사가 취해야 할 액션 암시

### 필수 포함 주제:
- 체험판 제공과 성과의 상관관계
- 멀티플레이 vs 싱글플레이 트렌드
- 리뷰/평가와 순위의 관계
- 지역별 유저 분포 (중국어권 등)
```

## 📝 프롬프트 4: TOP 5 게임 (04_top5_games.csv)

```
TOP10 게임 종합 평가 데이터에서 **상위 5개 게임 정보**를 추출해주세요.

### 출력 형식 (CSV):
rank,name,genre,wishlist_increase,wishlist_percent,review_status
1,{게임명},{장르},{찜 증가량},{증가율%},{리뷰상태}
2,{게임명},{장르},{찜 증가량},{증가율%},{리뷰상태}
3,{게임명},{장르},{찜 증가량},{증가율%},{리뷰상태}
4,{게임명},{장르},{찜 증가량},{증가율%},{리뷰상태}
5,{게임명},{장르},{찜 증가량},{증가율%},{리뷰상태}

### 게임명 매핑 (URL에서 추출):
- app/3576170 → 빈딕투스: 디파잉 페이트 (NEXON)
- app/3504780 → 와일드 게이트
- app/2841820 → Jump Ship
- app/2827200 → MIMESIS
- app/3763830 → Zoochosis
- app/2373990 → 나 혼자만 레벨업: 어라이즈 (Netmarble)
- app/3105890 → PIONER
- app/3640000 → Holstin

### 증가량 계산:
- wishlist_increase = 참여 후 찜 수 - 참여 전 찜 수
- wishlist_percent = (증가량 / 참여 전 찜 수) × 100
```

```

- 체험판 차트 진입 비율 (인기 체험판 차트 ÷ 전체)
```
