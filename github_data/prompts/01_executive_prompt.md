# 🎯 핵심 요약 (Executive Summary) - AI 인사이트 프롬프트

## 📋 개요
이 프롬프트는 대시보드의 **첫 번째 페이지 "핵심 요약"**에 표시될 AI 인사이트를 생성합니다.

---

## 🎯 인사이트 생성 목표
Steam Next Fest(SNF) 데이터를 분석하여 **게임 개발사/퍼블리셔가 다음 SNF를 준비할 때 참고할 핵심 인사이트 4개**를 도출합니다.

---

## 📊 입력 데이터 (raw/01_executive/)

| 파일명 | 내용 | 활용 방법 |
|--------|------|----------|
| `02_kpi_cards.csv` | 총 참가 게임, 찜 수 증가, 차트인 횟수, 멀티플레이 비율 | 전체 SNF 규모와 성공 지표 파악 |
| `04_top5_games.csv` | TOP 5 게임의 이름, 장르, 찜 수 | 성공 게임의 공통점 분석 |
| `05_chart_summary.csv` | 차트 진입 게임 수, 최다 차트인 게임, 체험판 비율 | 차트 성과 요약 |
| `06_genre_distribution.csv` | 장르별 비율 (액션 RPG, 슈팅, 로그라이크 등) | 인기 장르 트렌드 파악 |

---

## 📝 프롬프트

```
당신은 Steam Next Fest 데이터 분석 전문가입니다.

아래 CSV 데이터를 분석하여 **게임 개발사와 퍼블리셔가 다음 SNF를 성공적으로 준비하기 위한 핵심 인사이트 4개**를 도출해주세요.

### 분석할 데이터:
[여기에 02_kpi_cards.csv, 04_top5_games.csv, 05_chart_summary.csv, 06_genre_distribution.csv 내용 삽입]

### 출력 형식:
정확히 아래 CSV 형식으로 출력해주세요. 다른 텍스트 없이 CSV만 출력하세요.

id,icon,title,description,border_color
1,,{제목1},{설명1},#0047AB
2,,{제목2},{설명2},#3B82F6
3,,{제목3},{설명3},#8B5CF6
4,,{제목4},{설명4},#F59E0B

### 인사이트 작성 가이드라인:
1. **title**: 7단어 이내의 임팩트 있는 제목 (예: "체험판이 성공의 열쇠")
2. **description**: 데이터 기반의 구체적인 설명, 50자 내외
   - 반드시 수치를 포함할 것 (예: "TOP 10 중 80%가...", "92%가...")
   - 실행 가능한 조언 포함
3. **border_color**: 순서대로 #0047AB, #3B82F6, #8B5CF6, #F59E0B 사용

### 인사이트 주제 예시:
- 체험판 제공의 중요성
- 특정 언어 지원의 필요성
- 멀티플레이 vs 싱글플레이 트렌드
- 인기 장르 트렌드
- 한국 게임의 성과
- 차트 진입 전략
```

id,icon,title,description,border_color
