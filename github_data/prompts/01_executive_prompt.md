# 🎯 핵심 요약 (Executive Summary) - AI 인사이트 프롬프트 v2.1

## 📋 개요
이 프롬프트는 대시보드의 **첫 번째 페이지 "핵심 요약"**에 표시될 다양한 AI 인사이트를 생성합니다.

---

## ⚠️ 중요: CSV 포맷 규칙

### 필수 준수 사항
1. **숫자에 쉼표 금지**: `716572` (O) / `716,572` (X)
2. **큰 숫자는 K/M 단위 사용**: `+70.7K`, `+1.5M`
3. **퍼센트는 % 포함**: `50%`, `+25.6%`
4. **파이프(|) 구분자**: 여러 항목은 파이프로 구분
5. **빈 값은 빈 문자열**: `,,` (O) / `,null,` (X)
6. **모든 행 완전성**: 헤더 컬럼 수 = 데이터 컬럼 수

### 금지 사항
- 셀 내 쉼표 사용 (CSV 파싱 오류 유발)
- 줄바꿈 문자
- 따옴표로 감싸기 (불필요)

## 📝 CSV 1: 전략 카드 (01_strategies.csv)

### 스키마
```csv
id,icon,title,description,details
```

### 컬럼 설명
| 컬럼 | 타입 | 설명 | 예시 |
|------|------|------|------|
| id | 숫자 | 순번 1-3 | `1` |
| icon | 이모지 | 전략 아이콘 | `🎮` |
| title | 문자열 | 전략 제목 (15자 이내) | `체험판 필수 제공` |
| description | 문자열 | 한줄 설명 (25자 이내) | `체험판이 찜 수 증가의 핵심` |
| details | 파이프 구분 | 4개 세부항목 | `항목1\|항목2\|항목3\|항목4` |

### 출력 예시
```csv
id,icon,title,description,details
1,🎮,체험판 필수 제공,체험판이 찜 수 증가의 핵심,TOP 10의 50%가 체험판 제공|SNF 최소 2주 전 체험판 준비|1~2시간 플레이 분량 권장|버그 없는 안정적 빌드 필수
2,👥,멀티플레이 요소 추가,협동/경쟁이 바이럴 효과 극대화,TOP 10의 70%가 멀티플레이|Co-op 모드 친구 초대 효과|PvP 요소로 스트리머 관심 유도|싱글이라면 리더보드라도 추가
3,🌏,중국 시장 공략 필수,리뷰 언어 1위가 간체 중국어,92%가 중국어 간체 지원|Bilibili와 Weibo 마케팅|중국 스트리머 협업 고려|번체 중국어도 함께 지원
```

## 📝 CSV 3: 주요 발견점 (03_insights.csv)

### 스키마
```csv
id,icon,title,description,border_color
```

### 컬럼 설명
| 컬럼 | 타입 | 설명 | 예시 |
|------|------|------|------|
| id | 숫자 | 순번 1-4 | `1` |
| icon | 이모지 | 인사이트 아이콘 | `🎮` |
| title | 문자열 | 제목 (10자 이내) | `체험판이 성공의 열쇠` |
| description | 문자열 | 상세 설명 (60-100자) | `TOP 10 중 5개 게임이...` |
| border_color | HEX | 테두리 색상 | `#0047AB` |

### 작성 규칙
- **title**: 7단어 이내 임팩트 있는 제목
- **description**: 
  - 반드시 구체적 수치 포함 (예: "TOP 10 중 5개 게임이...")
  - 왜 중요한지 이유 설명
  - 개발사가 취해야 할 액션 암시

### 필수 포함 주제
1. 체험판 제공과 성과의 상관관계
2. 멀티플레이 vs 싱글플레이 트렌드
3. 리뷰/평가와 순위의 관계
4. 지역별 유저 분포 (중국어권 등)

### 출력 예시
```csv
id,icon,title,description,border_color
1,🎮,체험판이 성공의 열쇠,TOP 10 중 40%가 체험판 제공. TOP 50에서는 52%가 체험판 보유. 핵심 콘텐츠를 담은 체험판이 유저 관심도를 크게 높입니다.,#0047AB
2,👥,멀티플레이가 대세,TOP 10 중 80%가 멀티플레이 지원. 협동/경쟁 요소가 SNF에서 강력한 경쟁력이 됩니다.,#3B82F6
3,⭐,긍정 리뷰가 증명,TOP 10 중 30%가 매우 긍정적 이상 리뷰. 출시 전 충분한 테스트와 피드백 수집이 완성도를 높입니다.,#F59E0B
4,🌏,중국어권이 핵심,TOP 10 중 40%가 간체 중국어 리뷰 보유. 중국어 UI 현지화와 중국 시장 마케팅이 필수입니다.,#8B5CF6
```
