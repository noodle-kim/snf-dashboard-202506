# 📋 SNF 결산 리포트 - AI 인사이트 프롬프트

## 📋 개요
이 프롬프트는 대시보드의 **네 번째 페이지 "SNF 결산 리포트"**에 표시될 AI 인사이트를 생성합니다.

---

## 🎯 인사이트 생성 목표
태그, 언어 지원, 커뮤니티 데이터를 종합 분석하여 **다음 SNF를 준비하는 개발사를 위한 실행 체크리스트 형태의 인사이트 4개**를 도출합니다.

---

## 📊 입력 데이터 (raw/04_report/)

| 파일명 | 내용 | 활용 방법 |
|--------|------|----------|
| `02_kpi_cards.csv` | 참가 게임 수, 평균 태그 수, 언어 지원 수, 커뮤니티 채널 수 | 전체 트렌드 파악 |
| `03_tags_analysis.csv` | 인기 태그, 태그별 사용 비율, 장르별 필수 태그 | 태그 전략 수립 |
| `04_language_support.csv` | 음성/인터페이스/자막별 언어 지원 비율 | 현지화 우선순위 |
| `05_community.csv` | 커뮤니티 플랫폼별 사용 비율, 지역별 권장 채널 | 마케팅 채널 전략 |

## 📝 프롬프트

```
당신은 Steam Next Fest 출시 준비 컨설턴트입니다.

아래 데이터를 분석하여 **다음 SNF에 참가하려는 게임 개발사가 반드시 준비해야 할 핵심 사항 4개**를 체크리스트 형태로 도출해주세요.

### 분석할 데이터:
[여기에 02_kpi_cards.csv, 03_tags_analysis.csv, 04_language_support.csv, 05_community.csv 내용 삽입]

### 출력 형식:
정확히 아래 CSV 형식으로 출력해주세요. 다른 텍스트 없이 CSV만 출력하세요.

id,icon,title,items
1,,{제목1},{항목1}|{항목2}|{항목3}|{항목4}
2,,{제목2},{항목1}|{항목2}|{항목3}|{항목4}
3,,{제목3},{항목1}|{항목2}|{항목3}|{항목4}
4,,{제목4},{항목1}|{항목2}|{항목3}|{항목4}

### 분석 포인트:
1. **필수 태그 전략**: Action(90%), Multiplayer(54%) 등 효과적인 태그 조합
2. **언어 지원 우선순위**: 영어(필수) → 중국어 간체(92%) → 한국어(70%) → 일본어(68%)
3. **커뮤니티 채널**: Discord(90%), YouTube(85%), X(80%) 필수
4. **지역별 마케팅**: 글로벌, 중국, 한국 등 타겟 시장별 채널

### 인사이트 작성 가이드라인:
1. **title**: 3~5단어의 명확한 카테고리명 (예: "다국어 지원", "커뮤니티 구축")
2. **items**: 파이프(|)로 구분된 4개의 체크리스트 항목
   - 각 항목은 15자 내외로 간결하게
   - 수치가 있으면 포함 (예: "중국어 간체 필수 (92%)")
   - 실행 가능한 액션 아이템으로 작성

### 카테고리 예시:
- 체험판 준비
- 다국어 지원
- 커뮤니티 구축
- 태그 전략
- 멀티플레이 고려
- 마케팅 채널
```
