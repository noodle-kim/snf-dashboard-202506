# 🎮 2025년 6월 Steam Next Fest 대시보드

## 📊 대시보드 개요
2025년 6월 Steam Next Fest 크롤링 데이터를 기반으로 2026년 2월 SNF를 대비한 종합 분석 대시보드입니다.

## 🚀 사용 방법

### 1. 대시보드 열기
- `dashboard.html` 파일을 더블클릭하거나
- 웹 브라우저에서 직접 열기 (Chrome, Edge, Firefox 등)

### 2. 페이지 구성
대시보드는 8개의 카테고리로 구성되어 있습니다:

#### 📈 Executive Summary
- **KPI 카드**: 총 참가 게임, 평균 찜 증가, 평균 차트인 횟수, 멀티플레이 비율
- **TOP 5 Quick View**: 상위 5개 게임의 찜 수와 증가량
- **장르 분포 차트**: 인기 장르 비율 분석
- **핵심 인사이트**: 전체 SNF 데이터 요약

#### 🎯 TOP 게임 분석
- **TOP 10 게임 상세 테이블**: 순위, 게임명, 장르, 리뷰, 데모 수, 찜 증가, 차트인 횟수
- **찜 수 증가 차트**: 상위 10개 게임의 찜 수 증가량 비교
- **장르 분포**: 액션 RPG, 슈팅, 공포 등 장르별 분포
- **인사이트**: 멀티플레이 우세, 차트 지속력 등

#### 📊 차트 성과 분석
- **총 차트인 151회** (47개 게임 참여)
- **차트 종류별 분포**: 인기 체험판, 인기 출시 예정, 떠오르는 출시 예정
- **TOP 15 차트 성과 테이블**: 게임별 차트 진입 횟수 및 최고 랭킹
- **일자별 차트 활동**: 6/10~6/16 기간 동안의 차트 활동 추이
- **인사이트**: 빈딕투스 14회 1위, 평균 3.2회 차트인

#### 🎨 장르 & 트렌드
- **18개 장르** 분석 (로그라이크 22%, 슈팅 16%, 시뮬레이션 12%)
- **멀티플레이 vs 싱글플레이**: 54% vs 46%
- **장르별 상세 테이블**: 게임 수, 비율, 대표 게임
- **2026 대응 전략**: 로그라이크 강력 추천, 멀티플레이 필수 검토

#### 📋 게임 상세 정보
- **평균 언어 지원**: 7.8개
- **한국어 지원률**: 70% (35/50개)
- **음성 지원률**: 42%
- **2025년 출시**: 82%
- **주요 퍼블리셔/개발사** 분석
- **출시일 분포 차트**

#### 🗣️ 언어 & 현지화
- **25개 이상 언어** 지원
- **필수 언어**: 영어 100%, 중국어(간체) 92%, 한국어 70%, 일본어 68%
- **현지화 전략**: 대형 퍼블리셔는 10개 이상, 인디는 2-5개
- **음성 지원**: AAA급 대부분 지원, 인디는 미지원

#### 🏢 퍼블리셔 분석
- **45개 이상 퍼블리셔**
- **한국 퍼블리셔**: NEXON (빈딕투스), Netmarble (나혼자레벨업)
- **인디 개발사 비율**: 72%
- **AAA 퍼블리셔**: 6개
- **시장 트렌드**: 셀프 퍼블리싱 증가, 중국 퍼블리셔 진출

#### 🇰🇷 한국 게임 현황
- **2개 게임** (빈딕투스 1위, 나혼자레벨업 6위)
- **평균 차트인 13회** (전체 평균 8.8회 대비 높음)
- **총 찜 증가 224K+**
- **성과 분석**: AAA급 품질과 글로벌 경쟁력 입증
- **전략적 시사점**: IP 기반 게임, Souls-like 장르 적중

## 📊 데이터 시각화
- **Chart.js 라이브러리** 사용
- **도넛 차트**: 장르 분포, 차트 종류별 분포, 멀티/싱글 비율
- **막대 그래프**: 찜 수 증가, 차트인 횟수, 장르 분포, 언어 지원
